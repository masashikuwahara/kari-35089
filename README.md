# MyArt
自分の好きな絵画、または描いた絵を投稿することができます

# アプリケーション概要
ユーザー新規登録後、共有したい絵を説明文とともに投稿することで
トップページに表示されます。
各詳細ページにコメント機能もあり、他のユーザが感想を言い合ったり
チャットもすることができます。
